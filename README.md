# TFM - Contribuyentes y Anomalías

Este repositorio contiene el código fuente en Python desarrollado para el Trabajo de Fin de Máster (TFM).  
El objetivo principal es la **simulación y detección de anomalías en contribuyentes**, aplicando técnicas de análisis de datos y gobierno del dato.

## 📂 Contenido
- `Simulacion_Contribuyentes.py`: Script principal con el código fuente del proyecto.

## ⚙️ Requisitos
- Python 3.9 o superior
- Librerías utilizadas:
  - pandas
  - numpy
  - matplotlib (si se generan gráficos)
  - scikit-learn (si se aplican modelos de detección)

Instalación rápida:
```bash
pip install pandas numpy matplotlib scikit-learn

#Ejecución:
python Simulacion_Contribuyentes.py

#Objetivos:
* Implementar un modelo reproducible para la simulación de contribuyentes.
* Detectar patrones y anomalías en los datos mediante técnicas estadísticas y de machine learning.
* Integrar el flujo de trabajo en entornos de BI para apoyar la toma de decisiones.
